Sistem back-end untuk manajemen perpustakaan, sudah di-deploy secara live.
- **Bahasa/Framework:** TypeScript, JavaScript, dibangun dengan **NestJS** dan **Prisma ORM**

- **Bahasa/Framework:** TypeScript, JavaScript, dibangun dengan **NestJS** dan **Prisma ORM**
